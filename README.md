# CARDCorrect
# Language: R
# Input: TXT (key, value pairs)
# Output: CSV
# Tested with: PluMA 1.1, R 4.0.0


CARD database corrected gene counts

Input - tab-delimited file of keyword-value pairs:
readcounts (TSV)
genelengths (TSV)
lateralcoverage (TSV)
readlengths (TSV)
metadata (CSV)
RDS (DIR)

Counts are output to CSV
